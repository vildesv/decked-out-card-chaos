# Decked Out: The Card Chaos

A **visual and interactive playing card demo** with shuffle, deal, and dark/light mode features. Perfect for showing off card animations and layouts!

### Features

- **Stacked deck animation** on page load
- **Shuffle cards** randomly around the page
- **Deal hands** to up to 4 hands (max 5 cards each)
- **Reset deck** functionality
- **Dark/Light mode** with glow effects
- Hover animations and card flipping
- Responsive design for mobile and desktop
